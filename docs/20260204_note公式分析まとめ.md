# note公式分析まとめ

note編集部の公式記事から抽出した「読まれる記事」「売れる記事」のノウハウ。
note-draft、note-reviewへの適用方針をまとめる。

---

## 1. タイトル設計（500件分析）

### 購入されやすいタイトルの3要素

1. **具体性**: 内容が明確に想像できる
2. **読者メリットの明確さ**: 何を得られるか一目瞭然
3. **トレンド性**: 今読むべき理由がある

### 実践ポイント

- 「誰に向けた内容か」を明示
- 「どんな価値があるか」を伝える
- 25文字程度にまとめる
- 「だれが」「何をして」「どうなったか」をオチまで含める

### NG例

- 曖昧・抽象的
- メリット不明瞭
- 期待させすぎ（読後の満足感とのギャップ）

**出典**: [有料記事500件を分析して見えた、noteで購入されやすいタイトル設計の基本](https://note.com/notemag/n/n367ec637ae54)

## 2. 無料エリアの書き方（100件分析）

### 購入につながる3要素

1. **共感・問いかけ**: 読者の悩みや違和感を言語化
2. **変化の物語**: Before-After、自分の経験を示す
3. **ベネフィットの提示**: 有料部分で何が得られるか明示

### 実践ポイント

- 冒頭で読者が自分の状況と重ねられる内容を配置
- 具体的な成果への期待を生む
- 「記事を購入すると何が得られるか」を明示

**出典**: [有料記事100件を分析して見えた、noteで購入されやすい無料エリアの書きかた](https://note.com/notemag/n/nb51c6c55f1ac)

## 3. 読みやすい記事の10のコツ

### 構成

- **逆三角形**: 冒頭に最も伝えたいメッセージ → 説明 → 補足
- **目次**: 長い記事には必須

### 読者体験

- 途中で読むのをやめない工夫
- 読者の目を意識した文章

**出典**: [読みやすい記事を書くための10のコツ](https://note.com/notemag/n/n0e4bc242c5b6)

---

## 4. 有料コンテンツを売る6つのヒント

1. **価値の明確化**: 無料部分で「何が得られるか」を示す
2. **著者情報の充実**: 信頼性・専門性をアピール
3. **価格理由の明示**: なぜこの値段か（納得感）
4. **他クリエイターの価格調査**: 市場相場を把握
5. **価格の調整**: 反応を見て変更可能
6. **キャンペーン活用**: 初期割引 → 通常価格

**出典**: [noteで有料コンテンツを売るための6つのヒント](https://note.com/notemag/n/na9fd8ce1a166)

## 5. ロングセラー記事へ育てる4つの方法

- 新しい情報や読者の声を追記
- 「最新情報がある」と伝わる工夫
- 過去記事も新規投稿と同程度に売れる
- ロングセラーは平均の20-30倍の売上になることも

**出典**: [すぐできる！有料記事を「ロングセラー記事」へ育てる4つの方法](https://note.com/notemag/n/nc176a6822f58)

## 6. 集客の3段階

1. **読者の目にふれる機会を増やす**: SNS拡散、プロモーション
2. **クリックしてもらいやすくする**: タイトル・概要文の工夫
3. **購入してもらいやすくする**: 価値提示

**出典**: [自分のnoteをもっと読んでもらうには？ 集客のアイデアまとめ](https://note.com/notemag/n/n6791ad15bb80)

## note-draftへの適用

記事ドラフト作成時に以下をガイドとして組み込む。

### 対話フロー追加

1. **テーマ確認時**
   - 「誰に向けた記事か」を明確化
   - 「読者が得られる価値」を言語化

2. **構成設計時**
   - 逆三角形構成を推奨
   - 目次の要否を確認

3. **タイトル案作成時**
   - 25文字目安
   - 具体性・メリット・トレンド性をチェック
   - 複数案を提示

4. **有料記事の場合**
   - 有料ラインの位置を提案（共感→変化→有料が王道）
   - 無料エリアに3要素が含まれているか確認
   - 価格設定の理由を明文化

## note-reviewへの適用

レビュー時のチェックリストとして組み込む。

### タイトルチェック

- [ ] 25文字以内か
- [ ] 具体的な内容が伝わるか
- [ ] 読者メリットが明確か
- [ ] 期待させすぎていないか

### 構成チェック

- [ ] 冒頭に結論・最重要メッセージがあるか
- [ ] 目次があるか（長文の場合）
- [ ] 逆三角形構成になっているか

### 無料エリアチェック（有料記事の場合）

- [ ] 読者の悩みを言語化しているか（共感）
- [ ] Before-Afterまたは経験が示されているか（変化）
- [ ] 有料部分で得られるものが明示されているか（ベネフィット）

### 著者情報チェック

- [ ] 専門性・信頼性が伝わるか
- [ ] 「なぜこの人が書くのか」が分かるか

### 価格チェック（有料記事の場合）

- [ ] 価格設定の理由が明記されているか
- [ ] 読者の納得感があるか

---

## 優先度

1. タイトル（最も重要。クリックされなければ読まれない）
2. 無料エリア（購入判断の決め手）
3. 構成（読了率に直結）
4. 著者情報（信頼性）
