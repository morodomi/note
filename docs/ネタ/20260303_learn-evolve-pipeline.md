# 自己学習パイプライン (learn/evolve)

## 角度

AIが自分のセッションログから失敗パターンを抽出し、次回以降の行動を自動改善する仕組み。
TF-IDFベースのパターン検出 + instinct蓄積 + skill自動進化。

## 差別化

- learn/evolveの具体的実装は未公開。
- TF-IDFという古典手法をLLMのスキル改善に使っている点が独自。
- 「AIが自分で学習する」は概念としては語られるが、具体的なパイプラインを公開している例は少ない。

## 構成案

1. 問題: 同じミスを何度もする。セッションが切れると学びが消える
2. learnスキル: セッションログからobserverエージェントがパターン検出
3. instinct蓄積: 検出されたパターンをinstinct候補として記録
4. evolveスキル: 蓄積されたinstinctからスキル定義を自動更新
5. 実例: 実際に検出されたパターンと、それによるスキル改善の具体例

## キーメッセージ

「LLMの自己改善にLLMは不要。古典的なTF-IDFで十分パターンは見つかる」
